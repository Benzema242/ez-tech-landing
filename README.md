# EZ Tech Solutions — Landing Page

Live site: [eztechbahamas.com](https://eztechbahamas.com)

## Deploy to Vercel

This is a static HTML site. Deploys instantly to Vercel.

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import this repo
4. Click Deploy (no build settings needed — it's static)
5. Add custom domain `eztechbahamas.com` in Vercel project settings

## Files

- `index.html` — main landing page
- `tweaks-panel.jsx` — design tweaks panel (dev only)
- `uploads/` — all images, video, logos
- `vercel.json` — caching config
